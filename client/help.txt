===HELP===
Useful Commands:
--clear : Clears chat window
--date : Replaces the --date command with the current date

--font : Changes the font according to arguments:
    FONT TARGET
    &selection : applies font to message enclosed by {}
    &message : applies font to entire message
    &all : applies font to this and all future messages
    no selection : applies font to message content following --font command
    FONT STYLES
    &bold : makes font target bold
    &italic : makes font target italic
    &color : applies the HTML color to font target
    &size : changes size of font target (1-7)

--help : Shows this help window
--newtab : Opens new tab in browser
--picture : Displays the picture of the url following the --picture command
--search : Opens a google search window for the message contents
--time : Replaces the --time command with the current time
--wolfram : Performs a wolfram query for the message contents. Displays text and image of result
